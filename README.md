# PubMetric CRM

**Your private, local, and AI-powered lead tracker.**

PubMetric is a lightweight CRM designed for individuals and small teams who prioritize privacy and simplicity. It runs entirely on your computer, meaning your data stays yours.

![App Screenshot](https://via.placeholder.com/800x450.png?text=PubMetric+CRM+Dashboard)

## ✨ Features

- **Local-First:** All data is stored in a private SQLite database on your device.
- **AI-Powered:** Integrated with Claude to draft follow-up emails and summarize notes.
- **Warm & Simple UI:** A "cozy" design that stays out of your way.
- **Cross-Platform:** Available for macOS, Windows, and Linux.

## 🚀 Installation

### macOS
1. Download the `PubMetric.dmg` file.
2. Open the `.dmg` file.
3. **Drag the PubMetric icon into your Applications folder.**
4. Launch PubMetric from your Applications folder.
5. *Note: You may need to right-click and select "Open" the first time due to macOS security settings.*

### Windows
1. Download the `PubMetric-Setup.exe` file.
2. Run the installer and follow the on-screen instructions.
3. Launch PubMetric from your Desktop or Start Menu.

### Linux
1. Download the `.deb` or `.AppImage` file.
2. Install via your package manager or make the AppImage executable.

## 🛠️ Setup

Once installed, follow the [SETUP.md](./SETUP.md) guide to connect your Anthropic API key and start tracking leads.

## 🔒 Privacy

PubMetric was built with privacy as a core value.
- No analytics or tracking.
- No cloud syncing (unless you manually back up your database).
- Direct connections to Anthropic for AI features.
